快速开始

# 1. 安装工具链
$ curl -fsSL https://get.lancy.fun | bash
# 2. 创建项目
$ lancy create my-project --template=webapp
# 3. 部署运行
$ cd my-project && lancy deploy --cluster=local
# 4. 查看状态
$ lancy status
terminal 正在部署...
✓ 容器镜像拉取完成
✓ 网络配置完成
✓ 服务启动成功

系统要求

  • • 支持虚拟化技术 (VT-x/AMD-V)
  • • 至少 4GB 空闲内存
  • • 20GB 可用磁盘空间
  • • Linux/MacOS/Windows WSL2

快速验证

运行以下命令验证安装是否成功：
$ lancy version
