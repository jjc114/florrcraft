为什么选择 LancyUI?

不同于通用的 Linux 发行版，我们剥离了繁重的驱动层，直接在微内核之上提供 WASM 运行时，让应用部署速度提升 3 倍。

核心架构特性

通过 YAML 格式定义资源配额，内核自动完成 CPU 与内存的硬性隔离，防止单个应用导致系统崩溃。
支持一次编译，在 ARM、x86 及 RISC-V 架构上无缝运行，内核自动处理指令集差异。

快速上手

$ aether install-toolchain
$ aether project create hello-world
$ aether run --target=local

提示：请确保您的设备已开启虚拟化支持 (VT-x/AMD-V)。
